# week14-finalsite

This is a Boards Site created to evaluate my skills, education and experience according to six objectives required in order to qualify to graduate with a Bachelor's degree in Digital Marketing. 

This site was created for CSC104 according to the following guidelines and standards for the final project: 

Final Project Overview
If you've made it to this point, your Final Project should be ready for submission. You will submit your final website now.

Final Project Guidelines
The Final Project will be graded on three components: 

The Website project itself with all the technical requirements.
The progress milestones leading up to your completed website.
The appearance and design of your Website. 
Technical Requirements
Project contains 3 pages (no more than 7) dedicated to your topic.
At least 3 subcategories (meaning, menu items such as About, Products, Gallery, Contact).
Your pages must have unique titles (unless it's an inline design). 
Images and text on every page.
External CSS file that controls layout
Site functions in all browsers (Firefox, Chrome, Safari, and Edge)
Incorporate some of the topics we've covered in class:
Appropriate meta tags.
One e-mail hyperlink on the contact page.
Consistency in design from page to page.
Designed for your target audience.
Specialty fonts. 
Multimedia. 
One page containing a form or a table.
Submission:
Submission Requirements

Please upload your final project to GitHub. Include the HTML files (index and other pages), the CSS files, any images and multimedia, and all other documentation of the site, as needed. 

Include a description of what the page(s) output in a readme.txt.
Include the page name in your readme.txt file.
Use well-formatted, easy-to-read, and commented code.
User-friendly interface, including readable output, Interface is self-explaining.
View the website using the GitHub HTML previewer: GitHub & BitBucket HTML Preview Links to an external site. 
Submit a YouTube video link or screenshot(s) of the application in action!
NOTE: * If you complete your final project fulfilling all the technical requirements, that is a passing grade, and it doesn't necessarily mean that you will be getting the highest grade. Pay close attention to all the requirements, not just the technical ones. For the grade of A your website has to be stellar based on the design principles we discussed in class.
